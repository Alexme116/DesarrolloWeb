Tags

<html>

Este tag se utiliza para la creacion de un documento tipo HTML.

<head>

Este tag se utiliza para establecer el titulo, estilos, entre otras cosas importantes del DOM.

<body>

Este tag se utiliza para encapsular todo lo que va a mostrar la pagina web.

<title>

Este tag se utiliza para asignar el titulo del DOM o pagina web.

<h1> ... <h6>

Este tag se utiliza para agregar un texto de estilo encabezado decrementando su tamaño dependiendo del numero utilizado despues del h de 1 a 6.

<p>

Este tag se utiliza para crear un texto de tipo parrafo.

<b>

Este tag se utiliza para poner el texto en negritas.

<br>

Este tag se utiliza para hacer un salto de linea.

<hr>

Este tag se utiliza para crear una linea horizontal del ancho total.

Links

Los links son creados con el tag de <a> y estos son utilizados para que el usuario haga CLICK sobre ellos y asi redireccionarlos a otra pagina.

Se deben de utilizar URL's absolutas si se quiere redireccionar a una pagina de otro sitio web como: "https://www.google.com".

Se deben de utilizar URL's relativas si se quiere redireccionar a una pagina dentro del mismo sitio web como: "index.html".

Tambien se pueden utilizar las id para trasladarse entre secciones dentro del sitio web por ejemplo: si <body> tiene id="top" utilizamos la referencia de "#top".

Images

Para poder agregar una imagen se debe de utilizar el tag de <img>.

El link utilizado igualmente puede ser una URL absoluta si se utiliza una imagen de internet, o una URL relativa si se utiliza una imagen dentro del dispositivo.

CSS

Para poder hacer uso de un archivo CSS se debe de llamar el archivo dentro del tag <head> utilizando el tag <link>.

Dentro del tag <link> debemos de definir el URL relativo del archivo CSS y asignando la realcion como "stylesheet"

JavaScript

Declaración de variables

Los tipos de declaración de variables son: Var, Let, y Const.

Tipos de datos

Los tipos de datos pueden ser: Undefined, Boolean, Number, String, BigInt, Symbol, Null, Object, y Function.

Bucles

Los tipos de bucles son: For, While, Do While, For in, y For of.

Asignación

Los tipos de asignaciones son: =, +=, -=, *=, /=, %=, y **=.

Declaración de variables

Var

Sirve para declarar una variable.

Let

Sirve para declarar una variable local a nivel de bloque.

Const

Sirve para declarar una constante, que será una variable con un valor de solo lectura.

React

Arrays

Los arrays son una estructura de datos que nos permite almacenar varios valores en una sola variable.
Esta es representada con el uso de brackets [ ]

Methods

Los arreglos y objetos utilizan metodos para poder manipular sus valores.
Por ejemplo: push, pop entre otros.

Objects

Los objetos son una estructura de datos que nos permite almacenar varios valores en una sola variable. Esta es representada con el uso de llaves { }

React Use

React te permite construir interfases de usuario a partir de componentes.
Logrando combinar los componentes para finalmente crear pantallas, páginas y aplicaciones completas.

Componentes

Los componentes de react son funciones de JavaScript.

Hooks

Los Hooks te permiten utilizar diferentes funciones de React de tus componentes.
Puedes utilizar los que contiene la libreria o crear los tuyos propios.
Por ejemplo: State y Effect Hooks.

State

Los States son utilizados como un tipo de memoria para los componentes.
Por lo que si quisieramos realizar un cambio en un componente necesitariamos el uso de un State para actualizarlo.

useState

El Hook useState te permite añadir estados a tus componentes funcionales.
Este utiliza dos variables:
el estado para mantener el dato y la funcion para actualizar el estado.
Por ejemplo:
const [index, setIndex] = useState(0)

useEffect

El Hook useEffect te permite realizar efectos secundarios en componentes funcionales.
Este se ejecuta despues de cada renderizado del componente.
